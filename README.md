# Flexgrid
Sass Grid using flexbox

## Install

Just add the _grid.scss to your project using Sass.

## Use the grid

This grid is based on container, children will fit

    <div class="grid-4">
	    <div>Hello, i'm 25% width</div>
	    <div>Hello, i'm 25% width</div>
	    <div>Hello, i'm 25% width</div>
	    <div>Hello, i'm 25% width</div>
    </div>
